Plans and Progress


Website Prototype

Our product will be built around a website that will allow potential students to be able to easily find the scholarships which they need in order to pursue their ambitions and join a university. It will aim to provide an easy-to-use interface where potential students can quickly search keywords or create a personalised account that allows them to view a range of scholarships that match each of their unique circumstances. It will be a one-stop shop where they can get all their information on the scholarships, they want to and can apply for without needing to conduct extensive research and wasting time researching scholarships which they have no chance in landing.

The website will use web scraping software to form a database of all the different scholarships that are available. The website will provide a tool for users to access this data and resolve their queries. The website will allow users to search for different scholarships using different predefined parameters to assist in the search, these include;

  • Locational Filters specifically Filtering By State
  • Filter By Specific scholarship type

The specific filters include;

  • Access and Equity Scholarships
  • Access to Accommodation scholarships
  • Achievement and Academic Based Scholarships
  • Scholarships for International study
  • Scholarships for Women
  • Scholarships for First Nation Peoples

With the use of Proto.io we created a prototype website that will be used to model the design of the actual website once its development begins. The current prototype website contains a home page, profile page, preferences page, search box and a simple login/sign up page.


The picture above shows the home page of the website that was designed, it was aimed to be kept simple but also allow for the users to have a quick experience wasting as little time as possible. From the home page, users can navigate to the search, settings, login and saved scholarships pages at a click of a button. They can also complete a quick search from the parameters shown above and find scholarships based on the top universities or by certain states in Australia. The home page will also contain an option where potential users can quickly sign up for the website and create an account the benefits of which are discussed below. This page will also contain reviews and user experiences which will help build trust and rapport with users by sharing the positive experiences of other users to give them comfort in trust in our website.

Web Scrapping Program

Two methods will be used to compile data on scholarships. For some websites, a python script can be used but as a result of some confusing coding conventions used on some web pages a WebScrape program can be used to better extract data. Both these methods will export the data into a CSV file. The CSV files will then need to be converted using Java Script into JSON then it can all be formatted using HTML and CSS in a way that makes it much easier to read for users.

What's left to do

For the program to become a polished final product the CSV files will need to be better formatted with certain parts of information such as page links needing to be removed from being viewed and then converted to a hidden hyperlink attached to the cell of the scholarship allowing users to be able to go straight to the webpage showing more information about the scholarship. There would also need to be a searching function that allows the scraped data to be sorted through by user preferences. We would also obviously need to code the prototype website and combine the CSV files with the scholarship information.

Besides from the more technical work, with most of our group members not being gifted in the arts we need to develop a logo which the scholafind product can be represented by. As none of our group members have experience in design, if the project was to continue this work will be outsourced and will be overseen by Tommy who has a marketing background to ensure that the design is feasible enough to be used with our product.
