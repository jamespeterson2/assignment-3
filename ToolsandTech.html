Tools and Technologies


For this project to be completed, we will need to request access to APIs (Application Programming Interface) from universities such as RMIT as an example, in order to access scholarship data however this may be unlikely due to privacy and security reasons. Because of this, we would need to utilize a web scraping program such as ScraperAPI which has 5000 API calls. This is a great tool for us as it is compatible with Mac and Windows which is required as our group members have both operating systems on their machines.


For the python script the code editor utilized is sublime text, this is used as it is a preference of many in the group with multi line editing being a powerful feature. Also, for python to work, the most up to date python needs to be installed as it has access to many importable libraries such as BeautifulSoup which is used to web scrape. For this we can use Anaconda Prompt and then in that prompt libraries can then be installed. The script will also be run through PowerShell on Windows machines.


As our python skills are limited because the course is an introduction to IT and many of the group members have little experience with python coding, for websites that are formatted in a way that makes it difficult to scrape data from them using python due to drop down menus that have varying tag names in the HTML and CSS a web scraper tool known simply as Web Scraper can be used as a Chrome extension to allow us to get as many scholarships as possible.

Testing

As we are creating a single website, we could have multiple testing periods where one component is added to a website and then is tested by the developer to use and analyse for any bugs or issues that may arise. Once tested, we will be able to move on and add the next component where we will be able to repeat the testing. After every part of the website has been added and tested, we will select a small number of students seeking scholarships to use our website and inform us on what went well and wrong. Once, feedback is received we will be able to amend the errors and publish the website publicly.
