Skills and Jobs


Skills required

- Programming language -

- Python scripting language -

- HTML & CSS -

- Inspecting webpage elements -

- Database management -

As web-scraping is software based, programming and coding skills will be highly valuable in this application, for this project the group has identified HTML, CSS and Python as the main skills required to achieve the project outcomes. HTML and CSS will largely be used to develop webpages structurally and visually for the website. It is an important skill as this will be directly judged by users as soon as they enter the site, as this forms part of the user experience (UX) and user interface (UI). Moreover, skills in Python programming language are as equally important, Python can be used to develop the back end of the project, making API, database and server functions and communications possible, which will ultimately allow the web scraper to function properly.


Positions:

Programmer:

The programmer’s duties include writing code needed for the web-scraping program to functions, they would also be involved in making sure everything is functioning well through maintaining, debugging, and testing systems and software within the project. Moreover, the role would entail technical support which can involve resolving user self-submitted problems and queries.


Data/Database Analyst:

The data/database analyst’s duties include designing, evaluating, reviewing, and implementing a database that would help organise collected data from the web-scraping application into databases to be then analysed and turned into meaningful and useful information. Furthermore, their role would include maintaining and updating the database to ensure smooth operation.


Business analyst:

The business analyst’s duties include examining the overall project activities and determining how better to achieve outcomes and communicating opportunities and findings back to the project team. This is supplemented with information technology components to analyse data and review project processes including the interaction between users and the website.


Web developer:

The web developer’s duties include designing and forming the website that will showcase and house the web-scraping application. This involves programming, developing, and designing the web pages that will a part of site’s functionality, including the navigation and overall user interface and experience. The role ideally would require someone who is proficient in front-end development but not limited to (i.e back-end development).
