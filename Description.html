Description


The project will involve the Development of a website in which the user can select a relevant university within Australia and have relevant scholarships and grants pop up based on eligibility criteria using data previously imputed by the user. The website will also have various filters in order to adjust the scope with which scholarships/grants that the user is eligible for are shown. Data imputed can be criteria such as:

Criteria/Hardships experienced GPA The user will be able to input their GPA based on a scale of 0.0 to 4.0 Level of Study so far (e.g., Undergraduate, Masters, PHD, None) As there are numerous levels of studies which each offer its own scholarships, the user can input what study they have completed so far to further refine the data presented to them. After the level of study is inputted, the user must also input the exact degree/certificate for which they have studied in order to refine the search.

Although there are databases that offer users to search for scholarships within Australia, none are as extensive and as personally catered to the user as the project proposed. It can be reasonably expected that the website, if implemented well, will become very successful among tertiary students looking to assess their eligibility for scholarships in Australia.
