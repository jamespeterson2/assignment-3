Overview


The plan of the project is to create a single database which offers the user the ability to input their own information in order to filter out any scholarships that the user is ineligible to receive, saving the user an extensive amount of time spent researching. A web scraping software will be used in order to collect the data required in order to offer the personalised experience, with the website being created on GitHub using HTML and CSS.

Motivation

Every single University within Australia offers numerous grants and scholarships to multiple applicants each year. Due to the large amounts of scholarships/grants on offer, it can be difficult and exhausting for an individual to find if they meet the eligibility criteria. The project will offer a single database in which users can research whether they are eligible for certain scholarships and grants without having to do extensive research. RMIT alone offers more than 2,000 scholarships every year, and with minute differences between each offered scholarships it can be exhausting for individuals to comb through evaluating their eligibility criteria.

Although there are databases that offer users to search for scholarships within Australia, none are as extensive and as personally catered to the user as the project proposed. It can be reasonably expected that the website, if implemented well, will become very successful among tertiary students looking to assess their eligibility for scholarships in Australia.
